Welcome to URL Shortening Microservice

To create a new shortened URL:

Please append a valid URL to the address of this page, followed by "/new/"

To follow a shortened URL created previously:

Append the shortened URL to the page address followed by a forward slash

Example use:

https://urlcomp.herokuapp.com/new/http://www.freecodecamp.com

produces the following shortened URL:

{ "urlExt": "c02fc4", "targetURL": "http://freecodecamp.com" }

and

https://urlcomp.herokuapp.com/c02fc4

redirects to http://www.freecodecamp.com
